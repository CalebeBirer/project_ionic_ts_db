#  Part1 ionic7-angular-sqlite-app

## Installation

```bash
git clone https://github.com/jepiqueau/blog-tutorials-apps.git
cd blog-tutorials-apps/Part-1/ionic7-angular-sqlite-app
npm install
```

## Run

```bash
ionic serve
```
